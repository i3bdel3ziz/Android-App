This app is made for students who wish to register in university by filling a simple form. 

The user can fill the information inside the "add student" button which will then save it into a local database ( SQLite ). 

As well as, the user can search the information of the form using the "search student" using the unique student ID, which then they can either delete or modify the form if they wish.

This app has used 7 activities that include MainActivity, AddData, Search, SQLiteHandler, PhotoActivity, My Widget, My Widget2.
